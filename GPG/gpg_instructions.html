How to encrypt data for sharing

Use case: you want to share a .csv/.xlsx/.sav dataset with a colleague. We will be using an open standard called GNU Privacy Guard to do the encryption. Read more on the techicalities here. Why should you encrypt and how it works? Check out this blog post!

Install GnuPG

  • Download GnuPG and install it.
  • Open the command line and type the following command to make sure it is working.
    gpg --help

How to use

Sender

  1. Ask to the reciver their public-key, then open the command line and import the sender's public key: 
    gpg --import name_of_pub_key_file
  2. check that the newly imported keys are in there: 
    gpg --list-keys
  3. Encrypt the file you want to share with their public-key:
    1. navigate to the folder where the file is: 
      cd path\to\folder\
    2. encrypt: 
      gpg --output myfile.txt.gpg --encrypt --recipient sender@senderdomain.com  myfile.txt

Receiver

  • Generate a GPG key-pair: 
    gpg --gen-key
  • Export public key to file: 
    gpg --armor --output mypubkey.gpg --export your.name@yourdomain.com
    and send it to sender.
  • Decrypt received encrypted file: 
    gpg --output myfile.txt --decrypt myfile.txt.gpg
